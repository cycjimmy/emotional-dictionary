# emotional-dictionary
An attempt to create an emotional dictionary

使用Word2Vec和SVM生成情感词典的尝试

## 安装依赖
```shell
# 创建虚拟环境
$ python3 -m venv venv

# 激活虚拟环境
$ source venv/bin/activate

# 安装依赖包
(venv)$ pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 退出虚拟环境
(venv)$ deactivate
```
